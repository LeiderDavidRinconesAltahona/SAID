## 📖 Descripción

**SAID (Sistema de Aprendizaje Interactivo Digital)** es una aplicación diseñada para apoyar a los aprendices en la gestión de sus actividades académicas mediante una plataforma interactiva y organizada.

El sistema permite registrar tareas, visualizar actividades en un calendario, administrar recordatorios y recibir notificaciones automáticas para mejorar la organización y el cumplimiento de responsabilidades académicas.

---

## 🎯 Objetivo General

Desarrollar una plataforma digital que facilite la organización académica de los estudiantes mediante la gestión eficiente de tareas, calendarios y recordatorios personalizados.

---

## ✨ Características Principales

### 👤 Gestión de Usuarios
- Registro de usuarios.
- Inicio y cierre de sesión.
- Recuperación de contraseña.
- Gestión de perfil.
- Actualización de información personal.

### ✅ Gestión de Tareas
- Crear tareas.
- Editar tareas.
- Eliminar tareas.
- Marcar tareas como completadas o pendientes.
- Asignar prioridades.
- Buscar y filtrar tareas.

### 📅 Calendario Académico
- Visualización mensual.
- Navegación entre meses.
- Consulta de tareas por fecha.
- Creación y edición de tareas desde el calendario.

### 🔔 Recordatorios y Notificaciones
- Configuración de recordatorios.
- Notificaciones automáticas.
- Historial de recordatorios.
- Múltiples alertas por tarea.

### 📊 Dashboard Principal
- Resumen de tareas.
- Actividades próximas.
- Notificaciones recientes.
- Acceso rápido a los módulos principales.
